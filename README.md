# Person Object Detection Snapshot System

## Overview
This project is a real-time Person Object Detection and Snapshot Capture System developed using YOLOv8, OpenCV, and Python.

The system detects persons from live webcam video, tracks detected individuals in real time, and automatically captures snapshots when a person is detected.

This project demonstrates the use of Artificial Intelligence, Computer Vision, and Deep Learning techniques for smart surveillance and monitoring applications.

- Real-time person detection
- Webcam-based live monitoring

- YOLOv8 object detection

## Project Structure
- `all.py` → Main detection and snapshot system
- `app.py` → Application execution file
- `det.py` → Detection processing module
- `main.py` → Main program controller

---

## How the Project Works
1. Webcam captures live video frames.
2. YOLOv8 processes each frame.
3. Persons are detected in real time.
4. Bounding boxes are drawn around detected persons.
5. Snapshots are automatically captured when detection occurs.
6. Real-time webcam feed is displayed on screen.
